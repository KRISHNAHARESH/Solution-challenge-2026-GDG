<h1>🧠 Problem Statement</h1><br>
<p>Every year, tons of food are wasted while many people go hungry. FoodLoop addresses this issue by creating a bridge between excess and need, ensuring food reaches the right people before it gets wasted.
Let's join hands and create a conservative food contribution organisation.</p><br>

<h1>📌 Overview</h1><br>
<p>FoodLoop is a smart platform designed to reduce food waste by connecting restaurants, hotels, and individuals with NGOs and people in need. 
The system enables efficient redistribution of surplus food in a timely and organized manner.</p><br>

<h1>🚀 Features</h1><br>
<ul>🔄 Food Donation System – Donate surplus food </ul>
<ul>📍 Location-Based Matching – Connect donors with nearby NGOs</ul>
<ul>⏱️ Real-Time Notifications – Instant updates for food availability</ul>
<ul>👥 User Roles – Donors, NGOs, and Volunteers</ul>
<ul>📦 Request & Claim System – NGOs can request available food</ul>
<ul>📊 Dashboard Analytics – Track donations and impact</ul>

<h1>🛠️ Tech Stack</h1><br>
<ul>Frontend: React.js / HTML / CSS / JavaScript</ul>
<ul>Backend: Node.js / Express.js</ul>
<ul>Database: MongoDB</ul>
<ul>APIs: REST APIs</ul>
